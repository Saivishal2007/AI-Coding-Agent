# AI Coding Agent 🚀

An AI-powered coding assistant capable of analyzing GitHub repositories, generating code, explaining code, debugging issues, and assisting developers through a modern web interface.

### Backend
- FastAPI
- Python
- Docker

### Frontend
- React
- TypeScript

### AI

- LangChain
- Vector Database

### DevOps

- GitHub Actions

- GitHub Integration
